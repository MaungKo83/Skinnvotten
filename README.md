En interaktiv Twine-fortelling basert på det kjente barneeventyret
Dette prosjektet er en digital, interaktiv gjenfortelling av Skinnvotten, laget i Twine (Harlowe).
Historien er bygget med tekst, bilder, lyd og tilpasset CSS/JS for å gi en mer moderne leseopplevelse.

📖 Om prosjektet
Formålet med prosjektet er å lære:

Hvordan lage interaktive historier i Twine
Bruke HTML, CSS og JavaScript inne i Twine-passasjer
Integrere bilder, lyd, valg og navigasjon
Lage en mer profesjonell og brukervennlig presentasjon

Eventyret handler om en mann som mister votten sin i skogen — og hvordan dyrene som bor der tar den i bruk.

🧰 Teknologi brukt
Prosjektet bruker følgende:

Twine (Harlowe 3.x) – motoren for historien
HTML – for bilder, lydknapper og struktur
CSS – for layout, farger, knapper og navigasjon
JavaScript – for tryggere navigasjon og lydavspilling
Egnet kunst/bilder i images/
Lydklipp i audio
