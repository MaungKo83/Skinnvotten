
🌲 Eventyret om Skinnvotten
Velkommen til Eventyret om Skinnvotten – en interaktiv fortelling laget i Twine (Harlowe).
Prosjektet visualiserer det klassiske norske barneeventyret gjennom tekst, bilder, lyd og moderne webteknologi.
👉 Live demo:
https://maungko83.github.io/Skinnvotten/

🧩 Innhold
Denne interaktive fortellingen inneholder:

Et strukturert eventyr bygget i Twine
Illustrasjoner og lyd til hver scene
Tilpasset CSS for et mer moderne og tydelig design
Egen JavaScript‑kode for sikker og stabil navigasjon
Responsivt design som fungerer på både PC og mobil

Designer.png

📖 Om prosjektet
Eventyret følger en mann som mister votten sin i skogen. Ett etter ett finner dyrene votten og kryper inn i den for å holde varmen.
Dette prosjektet ble laget for å trene på:

HTML i Twine‑passasjer
CSS for design, layout og navigasjon
JavaScript for trygg funksjonalitet
Strukturering av interaktive historier
Bruk av GitHub Pages for living av demo


🧰 Teknologi brukt

Twine (Harlowe 3.x)
HTML – bilder, knapper, lyd
CSS – layout, typografi, sidebar, effekter
JavaScript – navigation handling, lyd
GitHub Pages – publisering av demo
